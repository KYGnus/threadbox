About CFG Analysis
What is Control Flow Graph Analysis?

CFG analysis examines executable files to understand their execution flow, detect obfuscation techniques, and identify potential malware behaviors through:

  • Disassembly of executable code
  • Detection of dispatcher-based obfuscation
  • Analysis of indirect jumps and calls
  • PE structure analysis (sections, imports, exports)
  • Entropy analysis for packed code detection
Detected Patterns
  • Call/Jmp through registers (EAX, EBX, ECX, EDX)
  • Memory-based dispatchers
  • Dynamic target resolution
  • Anti-debugging techniques
  • Packed/obfuscated sections
Supports: Windows PE files (EXE, DLL, SYS)
